## What Viz Lab Does

Viz Lab is a visualization-oriented app for Cribl Search that helps transform query output into clear, interactive views. It combines charts, tables, and dashboard layout controls so you can move from raw events to a more usable operational view without constantly changing tools.

It is a strong fit for teams working with KQL, event fields, and result exploration. With options for visual customization, imported chart definitions, and collaborative workflows, Viz Lab provides a practical way to organize, compare, and present search data inside the Cribl App Platform.

## Capabilities

- 104 built-in visualizations for a wide range of charting and reporting needs
- Plugin gallery for extending charts with third-party extras
- Dedicated Events, Fields, and Chart result panes for focused analysis
- Field drilldown panel for faster inspection of values and context
- Drag-and-drop dashboard layout for arranging visual content
- Variables and tabs to support organized, repeatable views
- Import support for Splunk XML, Cribl JSON, and Viz Lab JSON
- Dark and light themes with per-visualization styling controls
- Drawing-tool annotations for marking up charts and dashboards
- CSV and NDJSON export for sharing or downstream processing

## How to Use Viz Lab

1. Run a KQL search in Cribl Search.
2. Open a visualization type from the available chart and table options.
3. Use the Events, Fields, and Chart panes to inspect results.
4. Drill into field values when you need more detail.
5. Arrange multiple views on a dashboard with drag-and-drop placement.
6. Add variables, tabs, and annotations to keep views organized.
7. Export output to CSV or NDJSON when you need to reuse the data elsewhere.

Example workflow:

- Start with a query result set
- Choose a chart, table, heatmap, or sparkline-style view
- Adjust theme and formatting options
- Save or share the dashboard layout for future use
